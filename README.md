# Argentinian Random Quote

## Usage

```js
const getRandomQuotes = require("argentinian-random-quote");

const quotes = getRandomQuotes(
    10, //Quantity of quotes (1 by default)
    true //If user wants repeated word or not (false by default) 
)
```

## How to colaborate
